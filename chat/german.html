Vajo Encrypted Secure Chat (VESCH) erfüllt weltweit einen dringenden Bedarf:

Eine simple Kommunikations-App, die schützt, was Sie in Ihrer privaten Kommunikation sagen.

Vajo Chat kann NICHT viele Dinge, sondern nur ein paar Dinge, die es hervorragend beherrscht. Ihre Nachrichten werden verschlüsselt, über unseren Server an den Empfänger gesendet, auf dem Telefon des Empfängers entschlüsselt und sobald der Empfänger die Nachricht erhält, wird sie für immer von unserem Server gelöscht. Die Nachrichten befinden sich also nur noch auf Ihrem Telefon und dem Ihres Gesprächspartners.

Vajo Chat erlaubt nur zwei Arten von Dateien für die Übertragung als Anhänge: Bilder und PDFs.

Die Verbindung zwischen Ihrem Vajo-Konto und Ihrer Telefonnummer besteht nicht. Sie können einfach eine E-Mail erstellen, wo immer Sie wollen, und diese E-Mail verwenden, um ein KOSTENLOSES Konto zu erstellen, und schon sind Sie startklar.

Vajo Chat hat keine PC-Version, denn egal, was man Ihnen erzählt, diese Umgebungen sind schwer zu sichern.

Dies ist ein minimalistisches CHAT + IMAGES + PDFs Programm, das durch asymmetrische Verschlüsselung hochgradig geschützt ist, so dass die Nachrichten von niemandem in der Mitte gelesen werden können, außerdem werden sie nie auf dem Server gespeichert, die Option, Nachrichten zu sichern oder auf dem Server zu speichern, gibt es nicht einmal. Wir benutzen keine externe API für diesen Zweck.

Im Notfall können Sie Ihr Konto von unserem Server und Ihre Nachrichten von Ihrem Telefon in a few seconds in wenigen Sekunden löschen.

Derzeit unterstützt Vajo Chat keine Sprache, dies kann in Zukunft angeboten werden, WENN wir sicher sein können, dass dies unbedenklich ist.

Die kostenlose Version ermöglicht es Ihnen, unbegrenzt mit bis zu 10 Benutzern zu kommunizieren und 3 Gruppen zu bilden. Die Gratis-Version enthält Google-Anzeigen, die uns helfen, die Betriebskosten zu decken. In der PRO-Version gibt es keine Beschränkungen für Benutzer und Gruppen und keine Werbeeinblendungen.

Wie man Vajo Chat benutzt

Es ist kinderleicht. Sie tragen eine E-Mail-Adresse (jede beliebige) und ein Passwort ein und erstellen ein Konto, verifizieren Ihre E-Mail-Adresse und schon sind Sie fertig. Sie können sich jetzt anmelden. Sie können anhand des Benutzernamens nach einem Benutzer suchen und einfach 'Hallo' sagen. Wenn Ihr Gesprächspartner dies akzeptiert, beginnt die Chatsitzung. Sie können auch eine Gruppe erstellen und ihr Benutzer hinzufügen. Die Nutzung ist ähnlich wie bei jeder anderen Chat-App.

Sie werden sich vielleicht fragen, warum wir Ihre E-Mail-Adresse benötigen? Um Sie auf dem Laufenden zu halten oder Änderungen vorzunehmen. Damit wir Ihnen ein neues Passwort schicken können, wenn Sie Ihr Passwort verloren haben.
